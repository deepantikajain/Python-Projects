# Online Food Ordering System (Django)

A full-stack Online Food Ordering System built using Django, HTML, CSS, and Bootstrap.  
This project allows users to browse restaurants, view food items, search, add items to cart, manage quantity, and place orders.

---

## Features

- User Authentication (Login / Register)
- Multiple Restaurants
- Dynamic Food Menu
- Search Functionality (Food + Restaurant)
- Food Image Upload via Admin Panel
- Add to Cart
- Increase / Decrease Quantity
- Remove Items from Cart
- Cash on Delivery (COD)
- Order Placement
- Admin Panel Management

---

## Technologies Used

- Python
- Django
- HTML5
- CSS3
- Bootstrap 5
- SQLite Database
